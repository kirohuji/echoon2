# 导游口试训练系统需求文档

## 1. 文档目的

基于 `guide-exam-shadcn-web-source-no-deps-20260425` 现有实现，梳理系统的业务目标、功能需求、关键流程与数据约束，作为后续产品迭代与研发实现依据。

## 2. 产品定位与目标

- 面向导游外语口试备考用户，提供题库练习、模考演练、学习资产沉淀（收藏/生词）与个人学习看板。
- 聚焦 PC 端体验，强调结构化练习与高频题稳定提分。
- 通过「题库绑定 + 模式切换 + 持续练习」形成可追踪学习闭环。

## 3. 角色与使用场景

- 学员用户：按报考省份/语种绑定题库，进行题目练习与学习，查看模考成绩和个人学习数据。
- 典型场景：
  - 首次进入时完成题库绑定。
  - 在题库首页筛选并进入题目。
  - 在练习页进行听读、显示答案/翻译、快捷键操作。
  - 在个人中心查看记录、收藏、生词与偏好设置。
  - 在模考页查看试卷与近期成绩。

## 4. 功能需求

## 4.1 路由与页面导航

- 基于 hash 路由，支持以下视图：
  - `#/`：题库首页
  - `#/practice/:topicId`：题目练习页
  - `#/mock`：模考页
  - `#/profile`：个人中心
  - `#/member`：会员权益页
- 顶部导航提供题库、模考、会员权益入口，头像按钮进入个人中心。

## 4.2 题库绑定与配置

- 首次进入必须弹出「选择当前使用的题库」对话框。
- 配置项：报考省份、教材语种、考试类型、面试形式。
- 根据配置联动筛选可用题库列表；无可用题库时提示用户切换组合。
- 用户确认后写入本地存储，并用于全局内容展示。
- 已绑定用户可在个人中心设置中再次调整。

## 4.3 题库首页

- 展示当前题库标题及绑定信息（省份、语种、面试形式）。
- 支持训练模式切换：
  - 练习模式
  - 学习模式
- 支持关键词搜索（当前对「景点介绍」分类生效）。
- 景点介绍卡片区：
  - 展示标题、时长、难度、摘要、掌握度进度条
  - 支持收藏/取消收藏
  - 支持进入详情/开始练习
- 其他题型列表：
  - 展示题型总量、已看/未看进度
  - 支持跳转到对应题目练习

## 4.4 题目练习页

- 展示刷题进度（当前题号/总题数）和上下题切换。
- 核心练习区包含题目与参考答案两个语音交互块，支持：
  - 播放英文文本（浏览器语音合成）
  - 单词点击释义提示（词汇表 + 题目词扩展）
  - 中文翻译显示/隐藏
- 答案显隐规则：
  - 练习模式：按题单独切换显示答案。
  - 学习模式：默认显示，可按题收起答案。
- 支持收藏题目、加入/移除生词。
- 支持快捷键操作：
  - `Space` 播放题目
  - `A` 显示/收起答案
  - `T` 显示/隐藏翻译
  - `F` 收藏/取消收藏
  - `←` 上一题
  - `Enter`/`→` 下一题
- 题目扩展信息区：
  - 题目信息（时长、难度、掌握度、练习要求）
  - 关键词
  - 重点词汇与生词本操作

## 4.5 模考页

- 展示模考试卷类型（标准卷/强化卷）与建议时长、题量、考察重点。
- 展示最近成绩列表（日期、试卷、分数、薄弱点）。
- 展示模考配置（题库版本、省份、语种、面试形式）。
- 提供模考看板示意（题目预览、开始作答、计时）。

## 4.6 个人中心

- 左侧为个人信息与二级菜单，右侧为内容区。
- 子模块：
  - 概览：题库信息、核心指标（练习天数、最近模考、收藏、生词、记录、模式）
  - 活跃度：按周热力图
  - 练习记录：最近练习条目
  - 收藏题目：收藏列表与快速打开
  - 生词本：生词列表与移除
  - 设置：自动播放、题库绑定调整、只读配置信息

## 4.7 会员权益页

- 展示会员套餐（标准会员、进阶会员）及价格。
- 展示权益对比表（题库、录音、AI 批注等）。
- 展示当前绑定题库信息与服务说明。

## 5. 业务规则与状态管理

- 持久化与数据源：
  - 题库绑定配置：服务端 `user_binding_config`（登录后由 `GET /bootstrap` 拉取写入前端内存状态；可在设置中重新绑定）
  - `guide-exam-favorites` / `guide-exam-words`：前端 persist 缓存（可与后端资产接口同步演进）
  - `guide-exam-preferences`：练习偏好等（如自动播放）
- 进入练习页时应自动回到页面顶部。
- 开启「进入题目后自动播放」后，切题应自动触发题目播放。
- 同一时刻仅允许一个语音块处于播放状态。

## 6. 数据需求（当前为 Mock）

- 题库配置数据：省份、语种、考试类型、面试形式、题库状态。
- 题目数据：题型、标题、摘要、掌握度、关键词、提示词、词汇、追问等。
- 练习内容数据：题目英文/中文、答案英文/中文。
- 学习行为数据：练习热力、模考历史、练习记录、会员权益。

## 7. 非功能需求

- 可用性：界面信息层级清晰，操作入口可视。
- 性能：搜索使用延迟值，减少输入抖动导致的重渲染。
- 兼容性：语音播放需兼容不支持 `speechSynthesis` 的环境（降级高亮轮播）。
- 可维护性：组件化拆分（UI 原子组件、业务组件、数据集中管理）。

## 8. 当前范围与后续建议

- 当前实现以前端 Demo 为主，核心交互完整，后端能力尚未接入。
- 建议后续迭代：
  - 接入真实题库与用户账号体系
  - 增加录音上传、评分与 AI 点评闭环
  - 增加学习计划、错题本、多维统计
  - 完善移动端适配与权限体系
